Allows authors and moderators to change comment status directly from the email notification.

== Description ==

Now you don't have to login to your user every time you get comment notification, you can approve/trash the comment with just one click.
Allows authors and moderators to change comment status directly from the email notification.


== Installation ==

1. Upload `one-click-comments` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= It is secure? =

We assume that everyone who have access to your email account can have access to your website, so yes, it is secure.
We generate security token with different slat per website.
